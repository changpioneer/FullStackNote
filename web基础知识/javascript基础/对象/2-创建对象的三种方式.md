#创建对象的三种方式
##1.字面量的方式
```javascript
var per1 = {
    name:"tom",
    age:23,
    play: function () {
        console.log("王者荣耀");
    }
};
```

##2.调用系统的构造函数
```javascript
var per2 = new Object();
per2.name = "jerry";
per2.age = 12;
per2.play = function () {
    console.log("吃鸡");
}
```

##3.自定义构造函数方式
```javascript
function Person(name, age) {
    this.name = name;
    this.age = age;
    this.play = function () {
     console.log("跳一跳");
    }
}
```
